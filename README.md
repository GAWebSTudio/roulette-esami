# Roulette per gli Esami

PWA in React + Vite per estrarre casualmente le pagine da interrogare.

## Avvio locale

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy su Vercel

1. Carica questo progetto su GitHub.
2. Importa il repository su Vercel.
3. Deploy automatico.

## File importanti

- `public/slot.mp3` → audio del gioco
- `public/manifest.json` → manifest PWA
- `public/service-worker.js` → service worker
